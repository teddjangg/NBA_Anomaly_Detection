# NBA Anomaly Detection

A machine learning project that detects unusual NBA player performance profiles using unsupervised anomaly detection methods and interprets the results with explainable AI techniques.

## Overview

This project explores anomaly detection on NBA player statistics from the 2023 season.  
Rather than treating anomalies as simple errors, we examine whether statistically unusual players reflect:

- elite multi-dimensional performance
- rare player archetypes
- low-minute contextual profiles

We compare univariate and multivariate anomaly detection approaches and use explainable AI methods to understand why certain players were flagged.

## Objective

The main goals of this project are:

- detect anomalous NBA player profiles using statistical and machine learning methods
- compare simple univariate outlier detection with multivariate anomaly detection
- explain anomaly predictions using interpretable surrogate models and XAI tools
- show that anomaly does not necessarily mean data error, but can reflect rare basketball roles or exceptional performance

## Dataset

The dataset contains NBA player statistics from the 2023 season, including:

- age
- minutes_played_per_game
- rebounds_per_game
- assists_per_game
- points_per_game

## Methods

### 1. Univariate Outlier Detection

We first applied the IQR-based outlier detection rule to each variable separately.

**Finding:**  
The IQR method detected very few outliers and failed to capture rare player profiles that are only unusual when multiple features are considered together.

### 2. Isolation Forest

Isolation Forest was used as the main unsupervised anomaly detection model.

**Why used:**  
It can detect observations that lie in sparse regions of the feature space, making it more suitable for multivariate player profiles.

### 3. Local Outlier Factor (LOF)

LOF was used as an additional anomaly detection method for robustness checking.

**Purpose:**  
To compare whether similar anomalous players are identified under a different neighborhood-based detection approach.

### 4. Surrogate Classifiers

To interpret the anomaly labels generated by Isolation Forest, we trained supervised surrogate models such as:

- Logistic Regression
  To provide interpretability, we trained a logistic regression model as a surrogate classifier.
  The anomaly scores generated by Isolation Forest were converted into binary pseudo-labels (1 = anomaly, 0 = normal), which were used as the target    variable. This allows the logistic regression model to approximate the decision boundary of the anomaly detection model and provide interpretable     feature coefficients.
  
- Decision Tree
- Random Forest
- MLP Classifier

These models help approximate the anomaly boundary and identify which features most strongly separate anomalous players from normal ones.

### 5. Explainable AI

We used:

- **SHAP** for global and local feature contribution analysis
- **LIME** for local explanation of individual anomaly cases

These methods helped explain why specific players were classified as anomalies.

## Key Findings

### IQR Was Not Enough

Simple univariate IQR rules were not effective because basketball performance is inherently multivariate.  
A player may not look extreme in a single stat, but can still be highly unusual in the combination of multiple stats.

### Detected Anomaly Profiles

The project identified players such as:

- Giannis Antetokounmpo
- Nikola Jokić
- Damian Lillard
- Tony Bradley

These cases represent different types of anomalies:

- **elite multi-category superstars**
- **rare playmaking big-man profiles**
- **high-usage scoring guards**
- **low-minute contextual anomalies**

### Anomaly Does Not Mean Error

A key conclusion of this project is:

> anomaly ≠ error

Many flagged observations were valid and realistic player profiles.  
They were statistically unusual because they represented rare roles, elite performance, or contextual differences in playing time.

## Example Interpretation

### Giannis Antetokounmpo
Flagged due to a rare combination of high scoring, high rebounds, and meaningful playmaking.

### Nikola Jokić
Detected as anomalous because of his unusual mix of elite rebounds and extremely high assists for a big man.

### Damian Lillard
Flagged mainly for extreme scoring output and high offensive load.

### Tony Bradley
A contextual anomaly with very low minutes and very low per-game counting stats, showing that low-opportunity players can also appear anomalous.
